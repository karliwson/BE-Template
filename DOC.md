# Partial Documentation

## Setting Up

```bash
# Install dependencies
yarn install

# Seed database
yarn seed

# Run the project
yarn start
```

## Testing

```bash
yarn test
```

## Linting with ESLint

```bash
yarn lint
```

## Info

The API responses follow the [JSend](https://github.com/omniti-labs/jsend) specification.
